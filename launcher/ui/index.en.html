PACKAGE OPERATIONS

From diagnostics to safe transfer.

Every action is limited to this package's Compose project and volumes. Evidence and encrypted backups are created outside the extraction; diagnostics exclude tokens and administrator passwords.

Authenticated encrypted backup

Quiesces three volumes and local state, then protects them with AES-256-CTR + HMAC-SHA256 Encrypt-then-MAC. The passphrase is never stored; losing it is unrecoverable.

Restore one writer

If current state exists, PF07 first creates an external pre-restore backup with the same passphrase, stops the old writer, and replaces only exact PF07 volumes.

Controlled 1.0.1 → 1.0.2 update

Run this from a separately extracted new archive. PF07 verifies both build identities, quiesces the predecessor, and binds the preserved `SHOP_INSTANCE_ID` and three volumes to exactly one successor writer.

Optional HTTPS tunnel

Only the storefront and WordPress-authenticated admin routes are allowlisted. n8n, databases, the Docker API, logs, and evidence stay private; local mode remains available after tunnel failure or shutdown.

Package-scoped uninstall

Create any needed encrypted backup first. Other Compose projects, volumes, and files are never touched.
