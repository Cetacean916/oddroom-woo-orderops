PACKAGE OPERATIONS

Backup, restore, and transfer

Actions in this area target the PF07 Compose project and volumes only. Backups and evidence are created outside the extraction; diagnostics exclude tokens and administrator passwords.

Encrypted backup

Pauses three volumes and local state, then protects them with AES-256-CTR and HMAC-SHA256. The passphrase is not stored and must be kept separately.

Restore backup

If current state exists, PF07 creates a pre-restore backup, stops the current runtime, and replaces PF07 volumes only.

Transfer 1.0.1 state

PF07 verifies both package identities, stops the 1.0.1 runtime, and connects its `SHOP_INSTANCE_ID` and three volumes to this package.

HTTPS tunnel

Only the storefront and WordPress administrator routes are exposed. n8n, databases, the Docker API, logs, and evidence remain private; the local store remains available after the tunnel stops.

Remove package

Create any needed encrypted backup first. Other Compose projects, volumes, and files are never touched.
