# PF07 Windows KVM owner test kit

1. `PF07-KVM-TEST.html`을 더블클릭합니다.
2. 받은 buyer ZIP을 선택해 파일명과 SHA-256을 검증합니다.
3. 화면의 1~10번 순서로 추출, 필수 구성요소 복구, 허브, 상점/관리자, 상태, 중지, 재시작/복구, 진단, 증거 내보내기를 수행합니다.
4. 마지막 버튼으로 `PF07-WINDOWS-KVM-RESULT.json`을 저장합니다.

보조 자동 수집은 `RUN-KVM-TEST.cmd`를 더블클릭합니다. 파일 선택기로 buyer ZIP을 고르면 해시·한글/공백 추출·PE 제품 버전을 검사하고 바탕화면에 JSON을 저장합니다. 이 키트 자체는 실제 KVM 실행을 주장하지 않으며, owner가 수행한 체크만 결과 파일에 기록합니다.
