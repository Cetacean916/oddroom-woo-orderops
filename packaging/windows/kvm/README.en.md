# PF07 Windows KVM owner test kit

1. Double-click `PF07-KVM-TEST.html`.
2. Select the downloaded buyer ZIP and verify its filename and SHA-256.
3. Follow steps 1–10 for extraction, prerequisite recovery, hub, store/admin, status, stop, restart/recovery, diagnostics, and evidence export.
4. Use the final button to save `PF07-WINDOWS-KVM-RESULT.json`.

For assisted collection, double-click `RUN-KVM-TEST.cmd`. Choose the buyer ZIP in the file picker; it checks the hash, a Korean/space extraction path, and PE product version, then saves JSON to the desktop. The kit itself makes no actual-KVM claim; only checks performed by the owner are recorded.
