# OFFSET OrderOps · Windows x64

1. ZIP 전체를 하나의 폴더에 풉니다. 공백·한글이 들어간 경로도 지원합니다.
2. `PF07-Launcher.exe`를 더블클릭합니다.
3. 허브가 필수 구성요소를 확인합니다. 누락 시 표시되는 번호 순서와 공식 설치 링크를 따르고 런처를 다시 엽니다.
4. `런타임 시작`이 준비 완료가 될 때까지 기다린 뒤 `상점 열기` 또는 `관리자 열기`를 사용합니다.

기본 0-KRW 컨테이너 경로는 Moby 엔진을 사용하는 Rancher Desktop입니다. Docker Desktop은 수신자가 현재 약관상 적용 가능성을 직접 확인한 경우에만 선택 가능한 대안입니다. Python 3.10 이상도 필요합니다. 첫 설치와 컨테이너 이미지 취득에는 인터넷 연결이 필요합니다.

명령 대체 경로는 `START-PF07.cmd`와 `pf07.cmd`입니다. 예: `pf07.cmd status`, `pf07.cmd restart`, `pf07.cmd diagnostics`, `pf07.cmd evidence-export`. 허브에는 인증된 암호화 백업/복원, 통제된 업데이트, 외부 제공자 CLI를 사용하는 옵션 HTTPS 터널, 확인된 패키지 범위 제거도 있습니다.

이 Windows 산출물은 Linux 호스트에서 PE·스크립트·아카이브·공통 코어 계약을 검증한 패키지입니다. 실제 owner KVM 전체 스택/브라우저 실행 결과를 주장하지 않습니다.
